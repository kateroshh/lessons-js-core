Добрый день!
Меня зовут Екатерина. Я работаю в Розничном блоке разработчиком BI. В качестве BI инструмента мы используем целевое решение QlikView. QlikView – BI-платформа с ассоциативным поиском в оперативной памяти со встроенными средствами ETL, разработчик QlikTech.

QlikView – устаревшее решение, которое почти не поддерживается QlikTech. К сожалению, QlikView плохо поддерживает разработку на JS, поэтому нельзя сделать красивые элементы (например выплывающие меню) и другие доработки на HTML, CSS и JS.

На рынке появилось новое решение - Qlik Sense. Qlik Sense позволяет разрабатывать интерактивную отчетность с красивой анимацией используя HTML, CSS и JS. Основное преимущество Qlik Sense можно использовать его модель данных и на ней разрабатывать полноценные порталы (сайты).

Пример крутой визуализации https://sudirvpo.sigma.sbrf.ru/vpo-bsr-qs/extensions/agile-dashboard/agile-dashboard.html (доступ должен быть у всех сотрудников банка, ссылка сигма). Моя мотивация – разрабатывать отчетность, которая будет более удобная для заказчика и которой смогут гордиться мои руководители.

Я самостоятельно проходила курсы по JS и верстке сайтов, поняла что нужен наставник который сможет направлять меня в огромном потоке информации по этой теме. Нашла курсы от Сбербанка и подала заявку. Сейчас я параллельно в свободное время прохожу курс по Qlik Sense. Этот инструмент сильно отличается от QlikView, мне нужны дополнительные знания.

Буду рада присоединиться к группе :)
